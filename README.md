# Metacorn - VIP Studio

यह एक वेब ऐप है जो वीडियो फाइल्स (MP4, MKV, MOV, AVI) से डिजिटल मेटाडेटा और ट्रैकिंग सिग्नेचर को साफ़ (purge) करता है।

## फीचर्स
* 5GB तक की बड़ी फाइल्स को सपोर्ट करता है।
* मेटाडेटा को तेजी से हटाने के लिए FFmpeg का उपयोग करता है।
* सरल और उपयोग में आसान इंटरफेस।

## टेक्नोलॉजी
* Python
* Streamlit
* FFmpeg

## ज्ञात सीमाएं (Known Issues)
* यह ऐप अभी Streamlit Community Cloud पर होस्ट है, इसलिए इनएक्टिविटी पर यह 'sleep mode' में चला जाता है।

## कैसे इस्तेमाल करें
1. ऐप लिंक पर जाएं।
2. अपनी वीडियो फाइल अपलोड करें।
3. प्रोसेसिंग पूरी होने का इंतज़ार करें और फाइल डाउनलोड करें।
